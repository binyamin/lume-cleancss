# CleanCSS for Lume

A CleanCSS plugin for [Lume](https://lume.land), a static-site generator for
[Deno](https://deno.land).

## Usage

Import the module from https://deno.land.

```js
import cleancss from 'https://deno.land/x/lume_cleancss@v0.1.0/mod.ts';
```
